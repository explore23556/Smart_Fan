# Smart_Fan

## 架構圖
### 主要是使用ESP32開發版接收溫濕度感測器回傳的數據來控制PC Fan，然後再透過WIFI(STA Mode)自動連接至手機，使得後續能夠藉由手機網路將資料上傳至Firebase，並且手機APP也會即時監控Firebase的數據。
